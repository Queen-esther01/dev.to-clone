# Norebase Assessment

This is an assessment for a frontend engineer role @[norebase](https://norebase.com)
meant to assess my abilities to create a responsive website/application.
## Technologies

This assessment was created using React & TailwindCSS

## How To Run The Project

In the project directory, run:

### `npm install`

This will install all the project dependencies will be installed.

### `npm start`

This will run the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.
